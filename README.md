# QA Test Automation Framework

Framework base para automação de testes web utilizando Selenium WebDriver, xUnit e C#.

## Objetivo

Disponibilizar uma estrutura organizada para criação e manutenção de testes automatizados em aplicações web, seguindo boas práticas de automação e o padrão Page Object Model (POM).

## Estrutura do Projeto

### Drivers

Responsável pelo gerenciamento e inicialização dos navegadores.

### Pages

Contém os Page Objects utilizados pelos testes.

### Tests

Contém os cenários automatizados e casos de teste.

### Utils

Classes auxiliares para screenshots, leitura de configurações e métodos reutilizáveis.

### TestData

Arquivos de massa de teste utilizados durante as execuções.

### Evidence

Armazena capturas de tela geradas durante a execução dos testes.

### Reports

Diretório destinado aos relatórios de execução.

## Tecnologias Utilizadas

* C#
* .NET
* Selenium WebDriver
* xUnit
* Git

## Conceitos Aplicados

* Testes Funcionais
* Testes de Regressão
* Page Object Model (POM)
* Data Driven Testing
* Captura de Evidências
* Boas Práticas de Automação

## Status

Estrutura inicial do framework criada para evolução futura dos cenários automatizados.
